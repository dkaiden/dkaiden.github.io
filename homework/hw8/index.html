CD Collection Viewer

Explanation of AJAX

This page uses one AJAX call in the loadDoc() function. When the button is clicked, an XMLHttpRequest object is created. The open() method configures it with GET as the HTTP method and "cd_catalog.xml" as the URL. The true parameter makes it asynchronous. The send() method initiates the request. When the XML response arrives, the onload event triggers myFunction(), which parses the XML using responseXML, extracts CD data with getElementsByTagName(), and builds an HTML table by looping through each <CD> element. The table is inserted into the page using innerHTML, updating only that section without refreshing the whole page.

REST API Report

API: Dog CEO Dog API (from apilist.fun)

Response Format: JSON

Authentication: No API key required

Cost: Free, no usage limits

Use Case: Great for projects needing random dog images without authentication hassles.

Documentation: Dog CEO Dog API Documentation

The Dog CEO Dog API is a RESTful service that provides access to a large collection of high-quality, breed-specific dog images. It's designed specifically for developers who want to easily integrate dog photos into their applications without the hassle of managing their own image database or dealing with complex authentication systems. What makes this API really stand out is how easy it is to get started. You don't need to sign up for an account or deal with API keys - just hit the endpoint and you'll get back clean JSON with direct image URLs. It's perfect for adding some fun to pet apps, building quick prototypes, or even just testing your AJAX skills. The image quality is solid, the response time is fast, and since it's completely free, you can use it in class projects or small personal apps without worrying about hitting limits. For developers who want to work with real API data without the usual setup headaches, this is a great choice.
